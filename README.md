# 校园多功能管理器（学生端）

##项目简介

本项目是一个校园学习管理器，旨在解决学生如何预约借书，不知道课表等常见问题，也可进行下载课表、成绩查询、线上提交作业等校园学习服务，采用HTML、CSS和JS技术栈三件套实现，所有数据均存储在浏览器本地，注重隐私和安全，开箱即用。

##核心功能

1.**课表查看/自定义**
  *  二级分类体系：星期 -> 节次
  *  支持为课表填写授课教师、换课说明，皆在解决学生未做好突然被调课、被占课的课前准备
  *  支持下载课表
  *  支持自定义选修课
  *  提供多条件组合筛选

2.**成绩管理**
  *  提供分析优势科目、薄弱学科、待加强科目，有利于学生寻找提分方向
  *  支持查看历史成绩
  *  三级分类体系：学期 -> 考试类型 -> 学科
  *  提供多条件组合筛选

3.**作业管理**
  *  提供接收作业，线上提交作业平台
  *  教师批改结果一目了然，便于学生订正
     
4.**预约借书**
  *  提供线上预约借书平台
  *  支持续借
  *  查询图书平台支持关键词搜索（匹配书名关键词）
  *  支持一次借多本书
  *  可查看预约借书记录、取书和还书时间

5.**个人中心**
  * 支持上传头像、修改昵称、更换手机号和修改账号密码
  * 未读消息和已读消息一目了然
  * 提供帮助与反馈平台，有问题或意见，可进行反馈
  * 提供退出登录，以便于登录其他账号


## 技术栈
**前端三件套**： HTML,CSS,JavaScript

### 作业提交、借书预约申请提交，审核流程
1. 上传作业 -> 状态标记为“待批改”
2. 教师批改作业后 -> 状态标记为“已批改”，并显示“优秀”或“良好”或“合格”或“不合格”等级
3. 提交借书申请 -> 状态标记为“待审核”
4. 教师审核后 -> 状态标记为“待借出”
5. 借出书后 -> 状态标记为“已借出”
6. 还书后 -> 状态标记为“已还书”
